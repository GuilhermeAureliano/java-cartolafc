# Java - Cartola FC API

Uma interface em Java para consumir a API REST do Cartola FC, facilitando a integração de aplicações com os dados do fantasy game.

- Em breve.

## Tecnologias

- Java 21
- Spring Boot 3.4.4
- Spring Web (REST)
- Springdoc OpenAPI (documentação Swagger)
- Maven (build e gerenciamento de dependências)
- JUnit (testes automatizados)

## Como usar

Em breve: instruções de instalação e exemplos de uso.

## Executar com Docker

### Build e execução (Docker)

```bash
docker build -t cartolafc .
docker run --rm -p 8080:8080 cartolafc
```

### Build e execução (Docker Compose)

```bash
docker compose up --build
```
